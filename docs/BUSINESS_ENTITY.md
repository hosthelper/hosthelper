## 과세유형 영향 (간이과세자)

### 현재
- 연 매출 8,000만원 미만 가정의 간이과세자
- 부가세 신고: 연 1회 (1월), 업종별 부가가치율 적용
- 매입세액공제 제한적

### 전환 시점 (예측)
- 플랫폼 매출 = ₩10,000 × 청소 Job 수
- 8,000만원 / ₩10,000 = **8,000 Job / 년**
- 월 기준 약 **670 Job / 월** 도달 시 일반과세자 전환
- PRD §6 Public Launch 타겟(월 1,000 Job)에서 자동 전환 예정

### 일반과세자 전환 시 필요 작업
1. 부가세 분기 신고 (4회/년) 체제 전환
2. `apps/api/src/modules/pricing/pricing.service.ts` 부가세 계산 추가
3. 세금계산서(현금영수증 대신) 발행 흐름 구현
4. 청소사 정산 시 부가세 매입세액공제 자동 처리
