# MIX 6공식 — 헬퍼유니버스 적용 가이드

> 출처: "MIX 섞기의 6가지 공식" 인포그래픽("다른 것을 연결할 때 새로운 가치가 탄생한다")을
> 헬퍼유니버스 현 단계(외국인 100명·공실헬퍼·중개사 3개 라이브 딜)에 적용한 버전.

## 0. 핵심 통찰

헬퍼유니버스는 처음부터 MIX 사업이다 — 숙박 × 청소 × 교육 × 의료인접 × 부동산중개.
따라서 MIX 6공식은 "새로 배울 것"이 아니라 "지금 가진 딜을 재조합해 매출을 키우는 도구"다.
원칙: **이미 가진 것끼리만 섞는다. 6주 타이머(외국인 100명) 도는 중엔 새것 추가 금지.**

## 1. MIX 6공식 → 헬퍼유니버스 적용표

| # | MIX 공식 | 이미 있는 것 | 지금 섞을 새 조합 |
|---|---|---|---|
| 1 | 분야를 섞어라 | 숙박+청소+교육+의료인접+중개 | 외국인 송객(여행) × 부동산 양도양수 = "외국인 떠난 빈 숙소를 양도매물로" |
| 2 | 경험을 섞어라 | 김태욱 호스트 운영·매칭 경험 | 박정호 운영 + 최인후 모객 + 김태욱 중개 = 3자 약정(체결) |
| 3 | 인풋을 섞어라 | 8전문가·에너지나침반·MIX | RCC 초대장 + 8전문가 카피 = 외국인용 다국어 랜딩 |
| 4 | 사람을 섞어라 | 중개사협회 800명·구해줘홈즈·교수 | 지부장 + 양수인 회원 = 양도양수 매칭 엔진 |
| 5 | 익숙함+새로움 | 익숙:에어비앤비 / 새로움:비의료 컨시어지 | 외국인=에어비앤비(익숙), 회복고객=RCC(새로움) |
| 6 | 계속 섞고 실험 | Phase 0 자동화 스택 | 첫 딜 데이터로 다음 조합 실험(Phase 1) |

## 2. 가장 강력한 MIX 한 방 — 3개 딜을 한 깔때기로

```
[외국인 100명 입국]  ← 라이프헬퍼 (최인후 모객 / 박정호 운영)
        │ 7~8월 운영 데이터·매출
        ▼
[호스트 "이 사업 되네"]  ← 헬퍼아카데미·하우스헬퍼로 흡수
        │ 일부는 숙소 양도 의사 발생
        ▼
[공실헬퍼 양도매물 업로드]  ← 양도인 30만
        │ 열람권 결제 양수인 유입
        ▼
[중개사 지부장 양도양수 계약]  ← 양수인 30만 + 중개수수료
        │ 새 호스트 탄생
        ▼
[새 호스트 = 새 공간]  → 처음으로 순환
```

핵심: "외국인 여행 사업"과 "부동산 양도 사업"은 남에겐 별개지만,
**호스트라는 한 사람 안에서 연결된다.** 외국인 받던 호스트가 → 지치면 → 양도하고 → 새 호스트가 받는다.

## 3. 공식 — A+B+C+경험

```
A 에어비앤비 송객·외국인 모객 (익숙)
+ B 비의료 회복 컨시어지 (새로움)
+ C 합법숙소 양도양수 중개 (부동산)
+ 김태욱의 호스트 운영·매칭 경험
= "호스트 한 명의 라이프사이클 전체를 먹는 플랫폼"
```
남이 못 베끼는 이유: A·B·C 따로는 다 있지만, 한 호스트 안에서 순환시키는 경험은 김태욱만 가짐.

## 4. 호스트 → 양도매물 전환 카톡 시퀀스 (5단계)

- **T1 운영 종료 직후**: "다음 시즌 운영? 아니면 양도 고민?" 의향 확인
- **T2 양도 의향 시**: 무료 노출 + 양수인 매칭 + 지부장 계약 처리 제안 (정보 5가지 요청)
- **T3 정보 수령 후**: 등록 완료 + 마스킹·열람권 결제 후 열람·검증 양수인만 안심 고지
- **T4 매칭 시**: 지부장 연결 → 현장 확인 → 표준 양도양수 계약 (수수료 한도 내)
- **T5 계약 후**: 재유입 안내(공실헬퍼·라이프헬퍼·넥스트헬퍼) + 후기 요청 → 순환 완성

가드: "수익/합법/투자 보장" 0건, "정보 제공·매칭" 포지션, 중개수수료 "한도 내" 단서.

## 5. 리스크 가드

1. 공식 6 "계속 실험"의 오해 금지 — 새 조합 실험은 6주 타이머 끝난 뒤. 지금은 이미 가진 것 연결만.
2. 너무 많이 섞으면 분산 = 헬퍼유니버스 고질병. "한 호스트의 순환" 하나로 좁힌다.
3. 외국인 떠난 숙소 → 양도매물 전환 시 양도인 동의·정보 마스킹·합법성 확인 필수.
