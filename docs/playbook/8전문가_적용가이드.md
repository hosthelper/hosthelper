> 출처: "억만장자 비즈니스 보드" 스킬(호모지·카돈·고딘·게리비·벨포트·브런슨·케네디·로빈스 8인)을 헬퍼유니버스 현 단계(RCC 1호 고객 미달성)에 맞춰 한국어·B2C·의료법 가드 통과 버전으로 재해석.

## 2. 호모지 — RCC를 "Grand Slam 오퍼"로 재설계

### 가치 방정식
| 요소 | 강화 |
|---|---|
| 꿈의 결과 | "회복 1주를 호텔처럼·집처럼, 누구의 시선도 없이" |
| 체감 가능성 | "공간 컨디션 미달 시 100% 환불 + 대체 공간 즉시 수배" 보증 |
| 시간 지연 | "선결제 후 48시간 내 공간 매칭, 7일 즉시 시작" |
| 노력·희생 | "공간 추천·세팅·청소·픽업까지 우리가" |

### 스택 (1인형 100만에 쌓는 가치)
1. 침구·홈케어 기기 풀세팅 (가치 30만)
2. 1회 픽업/드롭 옵션 (가치 15만)
3. 비의료 도시락 3일치 외주 제휴 (가치 18만)
4. 담당 컨시어지 24/7 카톡 (가치 20만)
5. 재방문 10% 재구매권 (가치 10만)
→ **총 가치 ~193만 / 가격 100만**

### 조건부 보증
> "입주 첫날 공간 컨디션 미달 시 100% 환불 + 동급 이상 대체 공간 즉시 수배. 못 하면 7일 패키지 전액 무료."

### 오퍼 네이밍
"리커버리 컨시어지" → **"7일 비의료 리커버리 컨시어지(RCC) 패키지"** (시스템처럼 들리게)

## 3. 고딘 — 보랏빛 소 + SVM + 포지셔닝 선언문

### 보랏빛 소 (주목 한 줄)
> "한국 최초, **공간을 빌리지 않는** 리커버리 컨시어지. 호텔처럼 머물지만 호텔이 아닙니다."

### 최소 유효 시장 (SVM)
- 너무 넓음: "회복 환자"
- 보통: "성형외과 환자"
- ✅ **SVM**: **지방·해외에서 서울 강남권 성형·피부 시술 받으러 와 1주 머물 곳을 찾는 30~50대 여성 (1인 또는 보호자 동반)**

→ 첫 100명은 이 SVM에 올인. 확장은 그 다음.

### 포지셔닝 선언문 (전 채널 공통)
> "RCC는 '병원 옆이 아닌, 내 페이스대로의 회복 1주'를 원하는 지방·해외 환자를 위한 비의료 컨시어지입니다. 호텔의 형식과 단기임대의 경직성 사이, 둘 다 아닌 제3의 선택지를 제공합니다."

### 리트머스 테스트
- ICP가 "딱 나잖아"라고 말하는가?
- 非ICP가 "이건 내 게 아니야"라고 말하는가?
- 누군가 정확히 따라 말할 만큼 구체적인가?

## 4. 브런슨 — 밸류 래더 + 신청 퍼널

### 밸류 래더

| 레벨 | 가격 | 무엇 |
|---|---|---|
| 1. 리드 마그넷 | 무료 | "서울 회복 1주 체크리스트 PDF" (병원 동선·식사·약국·표현 가이드) |
| 2. 트립와이어 | 5만 | "RCC 공간 매칭 진단" (1회 상담 + 공간 후보 3개) |
| 3. **코어** | **100~500만** | **RCC 1주 패키지** (정원별) |
| 4. 프리미엄 | 700만+ | RCC 프리미엄 (전담 컨시어지 + 픽업 + 식음 큐레이션) |

### 하이티켓 신청 퍼널
```
인스타·블로그 콘텐츠 → 리드마그넷 옵트인 → 5일 소프오페라(카톡)
→ 신청 폼(증상·일정·예산 사전 검증) → 컨시어지 콜 30분 → 클로징
```

> 콜드 트래픽이 100만을 바로 안 결제한다. **무료 → 5만 → 100만 사다리**를 깔아야 자연스러워진다.

### 소프 오페라 5일 시퀀스 (카톡 변형)
- D1: 무대 설정 — 체크리스트 전달 + 다음 메시지 예고
- D2: 드라마 — "지방에서 강남 와서 호텔에서 룸서비스 노크에 시달린 첫 환자 이야기"
- D3: 깨달음 — "공간을 빌리지 말고, 공간 위에 얹는 서비스만 팔자" (메커니즘 도입)
- D4: 숨은 혜택 — 사례 + 보너스 5종
- D5: 긴급성 + CTA — "이번 주 자리 2개 남음"

## 5. 케네디 — 메시지·시장·매체 일치 + 부유층 5기둥

### 메시지·시장·매체 매칭
| 요소 | 설정 |
|---|---|
| 메시지 | "병원 옆 아닌, 내 페이스대로의 회복 1주" |
| 시장 | 위 SVM (지방·해외 시술 여성) |
| 매체 | 인스타 릴스 + 카카오톡 채널 + 네이버 성형 후기 카페 + 병원 데스크 안내 카드 |

### 부유층(=하이티켓 B2C) 5기둥
1. **배타성**: "주당 5팀만 받습니다" (실제 capacity)
2. **권위**: "호스트헬퍼 729-33-00827 · 변호사 검토 표준계약"
3. **결과**: "공간 컨디션 미달 시 100% 환불" 보증
4. **프리미엄 프레젠테이션**: 인스타·웹·카톡 톤 통일
5. **프리미엄 가격**: 사과 금지 — "100만은 호텔과 다른 가치"

### 11단계 케네디 세일즈 레터 (RCC 랜딩용 변형)
1. 헤드라인: "강남 회복 1주, 호텔 말고 RCC."
2. 리드: "병원 옆 호텔의 사방 시선과 룸서비스 노크에 지치셨다면…"
3. 스토리: "(고객 1호) 부산 → 강남, 첫 시술 후 일주일을 어떻게 보냈는지"
4. 오퍼: RCC 1주 패키지
5. 증거: 후기 + 변호사 검토 + 사업자정보
6. 스택: 호모지 보너스 5종
7. 가격: 정원별 100만~700만
8. 보증: 조건부 100% 환불
9. 희소성: 주 5팀
10. CTA: "카톡으로 진단 신청"
11. 추신: "이번 주 자리 2개 남았습니다"

## 6. 적용 타임라인

| 단계 | 기간 | 주력 전문가 | 결과물 |
|---|---|---|---|
| **Phase 0** | 이번 주 | 호모지 + 고딘 | Grand Slam 오퍼 + 포지셔닝 선언문 |
| **Phase 1** | 2~4주 | 브런슨 + 케네디 | 밸류 래더 + 신청 퍼널 + 랜딩 카피 |
| **Phase 2** | 1~3개월 | 게리비 + 카돈 | 콘텐츠 볼륨 + 후속 시퀀스 |
| **Phase 3** | 3~6개월 | 벨포트 + 로빈스 | 클로징 스크립트 + CEO 상태 관리 |

## 7. 리스크 가드 (한국·의료 인접 적용 시)

1. **8명 전부 미국 B2B 마인드** — 한국 의료 인접 B2C에선 **표현 검열 필수**. 호모지 보증 문구 그대로 옮기면 표시광고법·의료법 위반 위험. 우리 의료법 가드 통과 한국어 버전만 사용.
2. **카돈식 "10배 행동" 오용 = 환자 유인**. 아웃리치는 병원이 아니라 **고객(C-direct)** 에게.
3. **벨포트식 확신 톤** — 한국 정서엔 자칫 강압. **"합리적인 사람" 톤 중심**으로 조정.
4. **동시 적용 금지** — Phase 0~1의 4명(호모지·고딘·브런슨·케네디)만 먼저. 나머지 4명은 결과 데이터가 쌓인 뒤.
