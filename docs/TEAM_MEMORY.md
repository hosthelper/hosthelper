# 🧠 팀 학습 메모리 (TEAM MEMORY)

> AI 직원팀이 **잘된 것·안된 것·내린 결정**을 쌓아 다음 판단에 재사용하는 노트.
> ruflo의 자가학습(ReasoningBank)을 우리 규모에 맞춘 경량 버전이며,
> `CLAUDE.md`의 "결정 영속화(Queryable Organization)" 원칙과 이어집니다.
>
> **갱신 주체**: 매주 `weekly-report`(주보고)가 그 주의 결과를 반영. 큰 결정은 즉시 추가.
> **사용법**: 새 캠페인·결정 전에 이 문서를 먼저 훑어 과거 교훈을 재사용.
