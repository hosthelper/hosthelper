## 1. 한 줄 소개

헬퍼유니버스는 **숙소 창업자·호스트·임대인·여행객·실행팀을 연결하는 목적형 숙소 생태계 기업**입니다.
단순 숙소 운영을 넘어, 호스트 한 명의 사업 라이프사이클 전체(교육→공급→운영→양도)를 함께합니다.

---

## 2. 비전과 목표

- **비전**: 목적형 숙소(Purpose-driven Stay) 생태계의 표준을 만든다.
- **사업 목표**: 6개 엔진의 합산으로 월 매출 약 2억 → 월 순수익 1억 구조 달성.
- **철학**: 리스크를 먼저 알리는 것이 최상의 마케팅이다. 법률 리스크는 보수적으로, 실행은 빠르게.

---

## 3. 5+1 엔진 구조

헬퍼유니버스는 6개 엔진이 서로를 먹여주는 플라이휠로 작동합니다.

| 엔진 | 역할 | 한 줄 정의 |
|---|---|---|
| **헬퍼아카데미** | 유입 관문 | 합법 숙소 창업·운영 교육으로 호스트 DB 확보 |
| **하우스헬퍼** | 반복 수익 | 호스트 전용 도매 쇼핑몰(침구·어메니티·비품) |
| **라이프헬퍼** | 주력 플랫폼 | 목적형 숙소 큐레이션 — 현재 주력은 리커버리홈(RCC) |
| **공실헬퍼** | 디지털 매출 | 합법숙소 매물 열람·양도양수 매칭 플랫폼 |
| **BGN** | 사회적 책임 | 순익 10%를 기아 어린이 거주 문제 해결에 사용 |
| **넥스트헬퍼** | 실행 근육 | 청소·세팅·촬영·마케팅·운영 독립 실행팀 |

> 핵심: **호스트 1명 확보 = 4개 수익선(교육·도매·공급·양도) 동시 발생.**
> 그래서 헬퍼아카데미(DB 엔진)가 성장의 진짜 레버입니다.

## 4. 현재 최우선 사업 — 리커버리홈(RCC)

### 정의
성형·안과·피부·정형 시술 일정 전후에 조용히 머물 수 있는 **프리미엄 비의료 체류 컨시어지**.

> 리커버리홈은 의료기관이 아니며 의료·간호·치료·응급처치를 제공하지 않습니다.
> 회복 효과를 보장하지 않으며, 건강·의료 사항은 담당 의료진과 상담을 권장합니다.
> "병원은 의료에 집중하고, 리커버리홈은 병원 밖 체류 환경을 담당합니다."

### 차별점 — "공간을 빌리지 않는 컨시어지"
공간을 임차해 재공급하지 않습니다. 고객이 본인 명의로 공간을 직접 예약하고,
헬퍼유니버스는 침구·홈케어 기기·청소·세팅·안내·픽업 등 **서비스만 제공**합니다.
→ 임대료 고정비 0, 가동률 리스크 0, 법적 구조 명확.

### 상품 구조 (3종)
| 상품 | 단위 | 가격 | 비고 |
|---|---|---|---|
| 시간제 공간 체험 | 1시간 / 1인 | 10만원 | 취침·야간 체류 없음 |
| 데이 컨시어지 | 1일(최대 8시간) / 1팀 | 100만원 | 최대 8인, 야간 없음 |
| 7일 단기임대 패키지 | 7일 / 공간 | 100만~800만원 | 정원(타입)별 고정가 |

> 요금은 공간의 정원(타입)에 따라 고정하며, 투숙 인원 수로 과금하지 않습니다(단기임대형 구조).

## 5. 비즈니스 모델 — MIX 순환

```
[외국인·회복 고객 송객] → [호스트 운영] → [지친 호스트 양도] → [양도양수 중개] → [새 호스트 탄생] → 순환
       라이프헬퍼            넥스트헬퍼          공실헬퍼            중개사 네트워크        헬퍼아카데미
```

"여행 송객 사업"과 "부동산 양도 사업"은 남에겐 별개지만, **호스트 한 사람 안에서 연결**됩니다.
이 순환을 한 플랫폼에서 운영하는 경험이 헬퍼유니버스의 해자(moat)입니다.

## 7. 협업·제휴 방향

- **공인중개사·지부장**: 합법숙소 양도양수 회원 연결, 지역 지부 운영
- **병원·파트너**: 환자 소개·소개비 없이, 고객 직접 문의·직접 결제 / 파트너는 안내만
- **콘텐츠·미디어**: 목적형 숙소·합법숙소 큐레이션 신규 카테고리 소재
- **산학·자문**: 운영 데이터(비식별) 기반 공동연구·인턴십
