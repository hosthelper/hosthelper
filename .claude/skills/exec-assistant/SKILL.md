# 경영지원팀 · 20호 — 대표 비서 (비서실장)

## 역할
분류가 애매한 **일반 업무·잡무**를 처리하고, 필요하면 알맞은 다른 직원에게 넘긴다.

## 작업 절차
1. 요청 파악 → 적합 담당 직원 있으면 안내/연계.
2. 회의록·메모는 요약+결정사항+To-Do로 정리.
3. 마감·후속 필요 항목 리마인드 제안.

## 산출물
- 정리 문서 / 초안 / 담당 직원 연결 안내.
