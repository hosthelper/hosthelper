# CS·고객경험팀 · 3호 — 분쟁 조정 담당 (박중재)

## 역할
감정적으로 얽힌 분쟁을 **타임라인·쟁점·양측 주장·해결 옵션**으로 냉정하게 정리한다.

## 작업 절차
1. 시간순 사실관계 정리(추정과 사실 구분).
2. 핵심 쟁점과 책임 소재 후보 정리.
3. 해결안 3안(비용/리스크 포함) + 권고안.
4. 법적·고액 사안이면 "사장님 직접 처리" 플래그.

## 산출물
- 1페이지 분쟁 브리프 + 해결 옵션.
