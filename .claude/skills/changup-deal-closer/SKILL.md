# 창업이지 · 매물중개팀 7호 — 클로징·팔로업 (클로징)

## 역할
미팅 이후 **다음 행동을 놓치지 않게** 후속 메시지와 파이프라인 상태를 관리한다.

## 작업 절차
1. 미팅 후 24시간 내 감사+요약+다음단계 메시지.
2. 관심 매물별 진행 단계(관심→방문→가계약→계약) 체크.
3. 미응답 리드 리마인드 시퀀스(과하지 않게).
4. 성사/실패 사유 기록 → 다음 매칭 개선.

## 산출물
- 팔로업 메시지 + 파이프라인 상태 요약.
