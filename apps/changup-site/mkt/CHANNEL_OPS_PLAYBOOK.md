# 채널 운영 플레이북 — 당근·오픈챗·네이버카페 (주간 운영표)

이미 만든 자산(기사·카드뉴스·릴스·아웃바운드 스크립트)을 "언제 어디에 올릴지"로 고정.
하루 15~20분, 대표님 혼자 실행 가능한 최소 루틴. 정책 안전 원칙 준수.

## 정책 안전 3원칙
- 당근: 외부링크·연락처는 게시글 본문/프로필에만. DM 자동응답·도배 금지.
- 오픈챗: 광고 톤 금지, 정보 공유 톤. 링크는 공지에 1회 고정.
- 카페: 각 카페 규칙 확인 후 홍보 게시판에만. 정보성 본문 + 하단 1줄 CTA.

## 주간 운영표 (요일별 1개 채널 집중)
| 요일 | 채널 | 할 일 |
|---|---|---|
| 월 | 당근 동네생활 | 오늘 기사 요약 1건 게시(정보 톤) + 프로필에 사이트 링크 |
| 화 | 네이버 카페 | 기사 본문 + 카드뉴스 6장, 지역 자영업 카페 1곳 |
| 수 | 오픈챗 | 권리금 팁 1개 공유 + 무료 진단 안내(공지 고정) |
| 목 | 당근(양도인) | 폐업/양도 글 5건에 아웃바운드 댓글(SELLER_OUTBOUND 참고) |
| 금 | 인스타/유튜브 | 릴스 1개 업로드(유튜브 쇼츠 동시), 캡션 복붙 |
| 토 | 카페/오픈챗 | 사례형 콘텐츠(권리금 바가지 방지) + 질문 유도 |
| 일 | 대표 스토리 | STORY_REEL 스토리 릴스 1개(주 1회 브랜딩) |

## 매일 공통 (5분)
- 새 설문 리드 확인 → 있으면 1~2일 내 연락(스크립트는 done-view 안내와 동일).
- 반응 좋은 글 1개 저장 → 다음 주 변주 소재로.

## CTA 키워드 고정
- 매수 관심 = '정보' / 양도(매도) = '등록' / 커뮤니티 = '상생'.

## 성공 지표(주간)
- 설문 리드 신규 건수, 양도(등록) 문의 건수, 채널별 유입([유입:] 태그).
- 목표: 첫 4주 안에 설문 리드 누적 20건 + 양도 매물 3건.
