# 양도인(매물) 아웃바운드 확보 키트

목적: 매수 리드만 모으면 보여줄 매물이 없어 이탈한다. 공급(양도인) 확보가 최우선 병목.
포지셔닝: 우리는 매물을 무료로 등록받아 창업 준비자와 연결하고, 계약은 공인중개사와 진행한다.
컴플라이언스: "중개/부동산 중개" 단어 금지. 수익·매도가 단정 금지, "확인 필요" 안전장치.

## 어디서 찾나 (양도인이 이미 모여 있는 곳)
- 당근 동네생활·중고거래: "가게 양도", "권리금", "인수", "폐업 정리" 검색
- 네이버 카페(지역 자영업·상권): 점포 양도 게시판
- 배달/자영업 커뮤니티, 상가 임대 게시글(임대인이 양도 정보 앎)

## 1) 당근·카페 폐업/양도 글에 남기는 댓글 (공개, 정책 안전)
```
정리 준비 중이시면 매물 무료 등록만 해두셔도 됩니다.
창업 준비하는 분들과 연결해 드리고, 계약은 공인중개사와 안전하게 진행돼요.
등록·상담 비용은 없습니다. 관심 있으시면 '등록'이라고 남겨주세요.
```

## 2) 1:1 접촉 스크립트 (상대가 '등록' 반응 시)
```
안녕하세요, 창업정보 모임입니다.
넘기시려는 가게 정보를 무료로 등록해두시면, 조건 맞는 창업 준비자에게 먼저 안내드려요.
필요 정보는 업종·위치(동 단위)·보증금/월세·권리금 희망가·양도 사유 정도입니다.
공개를 원치 않으시면 익명으로만 조건 매칭하고, 연락처는 성사 직전에만 공유합니다.
수수료는 계약 성사 시 매도인 측에서만 발생하고, 준비 단계는 전부 무료예요.
```

## 3) 손실회피 훅 (양도인 모집 콘텐츠용)
- "가게 닫으면 권리금은 0원이 됩니다 — 닫기 전에 등록부터"
- "폐업 신고 전에, 넘길 수 있는지부터 확인하세요"

## 4) 하루 실행 루틴 (10분)
- 당근·카페에서 '양도/폐업' 글 5건 찾기 → 위 댓글 남기기
- '등록' 반응 오면 스크립트2로 조건 접수 → 설문 링크로 등록 유도
- 접수된 매물은 사이트 매물 등록(내부) → 매수 리드와 매칭

CTA 키워드: 양도(매도) 리드 = '등록'.
