# 유니 보이스오버 릴스 (권리금 적정선)

카드뉴스 6장을 유니 목소리로 한 장씩 설명하는 세로형(9:16) 릴스입니다.

## 파일
- `reel_gwolligeum.mp4` — 최종 릴스 (1080×1920, 약 61초, H.264 + AAC). 인스타/틱톡/쇼츠 업로드용.
- `audio/c1.mp3 … c6.mp3` — 카드별 유니 낭독 음성.
- `index.html` — 브라우저용 자동재생 릴스 페이지. (호스팅 시 `/reel/`)
  - "재생"을 누르면 유니 낭독에 맞춰 카드가 한 장씩 전환됩니다.
  - "영상으로 저장" 버튼: 브라우저에서 바로 영상 파일로 내보내기(동일 출처 호스팅 시 동작).

## 자동 합성 파이프라인
`.github/workflows/build-reel.yml` (GitHub Actions):
1. CRM(seed_audio)로 생성한 유니 낭독(카드별 6개)을 러너에서 내려받음.
2. 저장소의 카드 이미지(`../mkt/cardnews/info_gwolligeum_0N.png`)와 ffmpeg으로 합성.
3. 각 카드를 해당 낭독 길이만큼(6.4–14.7초) 표시, 0.3초 페이드인, 하드컷 전환.
4. 결과 MP4를 브랜치에 커밋.

재생성이 필요하면 워크플로 파일을 수정해 push 하거나 Actions에서 수동 실행하세요.

## 내용 (대본)
1. 커버 — 권리금 적정선 네 가지 기준
2. 권리금은 바닥·영업·시설로 나눠 근거 요구
3. 영업권리금 = 검증된 월 순이익 기준(6–12개월치)
4. 시설권리금 = 감가된 값, 노후 설비는 철거비 주의
5. 양수도 계약서는 문서로, 세금 미리 확인
6. CTA — 무료 설문(매물 추천)·매물 등록, 실제 계약은 개업공인중개사

가드레일: 수익·성공 보장 없음, "중개" 아님(정보공유·연결·매칭), 실제 계약은 개업공인중개사.
